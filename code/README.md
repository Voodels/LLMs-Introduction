# Code Examples

This directory contains practical implementations of Large Language Model concepts covered in the main tutorial.

## Structure

Each subdirectory focuses on a specific aspect of LLMs:

### 01. Tokenization
Learn how to break down text into tokens that can be processed by the model.

### 02. Embeddings
Understand how tokens are converted into numerical representations.

### 03. Attention Mechanisms
Explore the attention mechanisms that allow models to focus on relevant parts of the input.

### 04. Transformer Blocks
Build the core transformer architecture blocks.

### 05. Complete Model
Combine all components into a complete working model.

### 06. Fine-tuning
Learn techniques to adapt pre-trained models to specific tasks.

### 07. Inference
Run inference and generate text with trained models.
